# Проект базы данных Rail Logistics (PostgreSQL)

Проект для хранения данных по железнодорожным перевозкам.  
Включает в себя таблицы, тестовые данные и запросы для расчёта показателей (KPI).

## Что внутри

Папка `schema/`:
- `01_tables.sql` — создание таблиц.
- `02_constraints.sql` — связи и индексы.
- `03_seed_data.sql` — начальные данные (минимум).
- `03_seed_data_full_utf8.sql` — полные тестовые данные в UTF-8.
- `04_views.sql` — представления для расчётов.
